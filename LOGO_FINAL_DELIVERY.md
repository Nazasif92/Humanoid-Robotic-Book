## Logo Design Details

### Core Design Concept

The logo features a **central AI eye symbol** surrounded by three concentric **orbital rings** representing robotics, automation, and technology systems.

### Visual Elements

**Central AI Eye** (Robot Eye Symbol)
- Outer circle: Cyan gradient (technology, intelligence)
- Middle iris: Magenta (innovation, control)
- Pupil: White with highlights (awareness, precision)
- Creates a distinctive "futuristic AI gaze" effect

**Orbital Ring System**
- **Ring 1** (Outermost, Cyan): Represents technology & action
- **Ring 2** (Middle, Magenta): Represents automation & control
- **Ring 3** (Inner, Lime): Represents knowledge & learning

**Knowledge Nodes** (Green circles)
- 5 knowledge points positioned around the eye
- Represents documentation, learning, and information
- Each node has a subtle halo ring

**Automation Nodes** (Gold circles)
- Strategic pulse points on orbital rings
- Represents active automation and data flow
- Indicates real-time processing

**Connecting Pathways** (Gray lines)
- Vertical, horizontal, and diagonal pathways
- Represents neural networks and communication
- Subtle opacity for minimalist aesthetic

**Corner Accents** (4 corner dots)
- Light gray precision markers
- Emphasizes geometric, technical nature
- Adds balance and symmetry

### Color Palette

| Color | Code | Meaning | Usage |
|-------|------|---------|-------|
| **Cyan** | #00D4FF | Technology, action, knowledge | Rings, nodes, main structure |
| **Magenta** | #FF00FF | Intelligence, innovation, control | Inner eye, automation, focus |
| **Lime** | #00FF88 | Knowledge, growth, documentation | Knowledge nodes, learning |
| **Gold** | #FFD700 | Automation, value, precision | Pulse points, indicators |
| **White** | #FFFFFF | Core, clarity, precision | Eye pupil, highlights |
| **Light Gray** | #E0E0E0 | Structure, framework, support | Border, pathways, accents |
| **Transparent** | - | Professional appearance | Background |

**Advantages**:
- Bitmap format (instant rendering)
- Optimized file size
- Perfect for navbar
- Works in all browsers

**Placement**:
```
static/img/logo.png
```

**Docusaurus Configuration**:
```javascript
// docusaurus.config.js
const config = {
  presets: [
    [
      '@docusaurus/preset-classic',
      {
        docs: {...},
        theme: {
          customCss: require.resolve('./src/css/custom.css'),
        },
      },
    ],
  ],

  themeConfig: {
    navbar: {
      logo: {
        alt: 'Humanoid-Robotic-Book Logo',
        src: 'img/logo.png',
        srcDark: 'img/logo.png', // Same in dark mode
        height: 32,
      },
    },
  },
};
```

## Setup Instructions

### Step 1: Copy Logo File
```bash
# Copy PNG version (recommended)
cp logo.png static/img/logo.png

# Or copy SVG version
cp logo.svg static/img/logo.svg
```

### Step 2: Update Docusaurus Config
Edit `docusaurus.config.js` and add the logo configuration (see "How to Use" above).

### Step 3: Test Locally
```bash
npm run dev
# Visit http://localhost:3000
# Logo should appear in navbar top-left
```

### Step 4: Build & Deploy
```bash
npm run build
vercel --prod --yes
```

### Step 5: Verify
After deployment, check:
- Logo appears in navbar
- Logo is crisp and clear
- Logo responsive on mobile
- Logo works in dark mode (if configured)
