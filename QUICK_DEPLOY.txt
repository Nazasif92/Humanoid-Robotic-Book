DEPLOYMENT - CHOOSE YOUR METHOD
================================================================================

METHOD 1: VERCEL DASHBOARD (RECOMMENDED - NO TOKEN NEEDED)
-----------------------------------------------------------
1. Open: https://vercel.com/new
2. Click: "Import Git Repository"
3. Select: github.com/Nazasif92/Humanoid-Robotic-Book
4. Click: "Deploy"
5. Wait: 1-2 minutes
6. Done: Site is live!

TIME REQUIRED: 3 minutes
DIFFICULTY: Easy


METHOD 2: VERCEL CLI (WITH PERSONAL ACCESS TOKEN)
---------------------------------------------------
1. Get token: https://vercel.com/account/tokens
   - Click "Create Token"
   - Name: "Humanoid-Robotic-Book"
   - Expiration: 7 days
   - Copy token

2. Run in project directory:
   export VERCEL_TOKEN="your_token_here"
   vercel --prod --yes

3. Wait: 30 seconds
4. Done: URL appears in output

TIME REQUIRED: 2 minutes
DIFFICULTY: Easy


METHOD 3: GITHUB AUTO-DEPLOY (RECOMMENDED FOR CI/CD)
-----------------------------------------------------
1. Connect Vercel to GitHub account
2. Import repository
3. Every git push triggers auto-deploy
4. No manual deployment needed

TIME REQUIRED: 5 minutes (first-time setup)
DIFFICULTY: Easy


METHOD 4: VERCEL CLI (WITHOUT TOKEN - INTERACTIVE)
----------------------------------------------------
1. Run: vercel login
   - Opens browser for authentication
   - Follow prompts
   - Returns to terminal when complete

2. Run: vercel --prod --yes

3. Done: Site is deployed

TIME REQUIRED: 5 minutes
DIFFICULTY: Easy
